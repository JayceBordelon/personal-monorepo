# Daily-email + dashboard refresh — visual snapshots

Companion to **PR #41** (`feat/daily-email-content`). Captures every route at desktop (1440×900) and mobile (iPhone 14 Pro, 390×844), in **both light and dark themes**, after the full set of changes lands. Generated by `scripts/ux-audit/audit.mjs`.

This branch is screenshots-only and never merges; it exists so the PR description can hyperlink to a page GitHub renders inline.

> Audit summary at capture time: **0 contrast issues** across {light, dark} × {desktop, mobile}. Only flags remaining are checker false-positives (decorative gradient orbs, short-text link widths) and the deliberate 404 test path.

## Reproducing

```bash
git checkout feat/daily-email-content
cd vibetradez.com/local
docker compose -f docker-compose.local.yml up --build -d

cd ../../scripts/ux-audit
npm install   # first time only
node audit.mjs
# Output: scripts/ux-audit/output/{report.json,summary.md,screenshots/}
```
